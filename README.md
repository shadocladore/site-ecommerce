# site-ecommerce
## Présentation
Ce projet est un site e-commerce fonctionnel conçu pour la vente de produits en ligne. Il permet aux utilisateurs de :
- Parcourir le catalogue de produits
- Ajouter des produits au panier
- S’inscrire et se connecter
- Finaliser des commandes

## Technologies utilisées
- **Frontend :** HTML, CSS, JavaScript
- **Backend :** PHP
- **Base de données :** MySQL
- **Outils :** Git/GitHub, Trello, FileZilla pour le déploiement

## Objectif pédagogique
Ce projet sert de support pour un **TPE sur le cycle de vie logiciel**. Il permet aux étudiants de pratiquer :
1. Gestion de version avec Git/GitHub
2. Modification et personnalisation du site
3. Tests simples
4. Déploiement en ligne sur un hébergeur gratuit
5. Documentation et suivi du SDLC

## Déploiement
Le site peut être déployé sur n’importe quel serveur PHP/MySQL ou hébergeur gratuit (InfinityFree, 000webhost, etc.) via FTP (FileZilla).

## Structure du projet
- `index.html` : Page d’accueil
- `css/` : Styles CSS
- `js/` : Scripts JavaScript
- `php/` : Fichiers PHP pour backend et gestion des formulaires
- `images/` : Images du site
- `database.sql` : Script pour créer la base de données
